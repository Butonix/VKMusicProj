# VKMusicProj 2.0
VK music web app created with React and Redux.

## Features
+ show your audiolist
+ search audios
+ show recommendations
+ download audios

## Setup and Run App Locally
+ clone repo
+ `npm install`
+ `npm start`
+ go to `http://localhost:3000/` in your browser
